# Terraform-Beginner-to-Advanced
This repository contains a complete Terraform learning journey from basics to advanced level. It is designed for beginners and DevOps engineers to learn Infrastructure as Code (IaC) using Terraform with hands-on examples and real-world projects.

This course will be covered in 11 - SESSIONS, below are the session details

---

# SESSION - 1. Foundations of Infrastructure as Code (IaC)

## Topics
- What is Infrastructure as Code (IaC)
- Imperative vs Declarative approach
- Benefits of Terraform
- Terraform vs other tools (Ansible, CloudFormation, Pulumi)

---

# SESSION - 2. Terraform Basics

## Topics
- Installation
- Terraform workflow:
  - `terraform init`
  - `terraform plan`
  - `terraform apply`
  - `terraform destroy`
- Understanding `.tf` files

---

# SESSION - 3. Providers & Resources

## Topics
- Providers
- Multi-cloud concept
- Resource blocks
- Data sources

---

# SESSION - 4. Variables & Outputs

## Topics
- Input variables
- Variable types and defaults
- `.tfvars` files
- Environment variables
- Outputs

---

# SESSION - 5. State Management (Very Important)

## Topics
- Terraform state
- Local vs remote state
- State locking
- State file structure

---

# SESSION - 6. Modules

## Topics
- What are modules
- Root vs child modules
- Module structure
- Terraform registry

---

# SESSION - 7. Functions & Expressions

## Topics
- Built-in functions
- Conditionals
- Loops:
  - `count`
  - `for_each`
- Dynamic blocks

---

# SESSION - 8. Provisioners & Null Resources

## Topics
- local-exec
- remote-exec
- null_resource

---

# SESSION - 9. Workspaces & Environments

## Topics
- Managing dev/stage/prod
- Workspaces vs separate state files

---

# SESSION - 10. Debugging & Troubleshooting

## Topics
- Logging (`TF_LOG`)
- Common errors
- Dependency issues

---

# SESSION - 11. Advanced Topics

## Topics
- Remote backends
- Terraform Cloud & Enterprise
- Sentinel policies
- Drift detection
- Import existing infrastructure

---

# 📚 Outcome

By the end of this course, students will:
- Understand Terraform deeply
- Build real-world infrastructure
- Be ready to crack DevOps interviews
